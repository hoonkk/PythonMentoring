1. 각자 자기소개 하기
	1)나 소개하기(학번 / 학과 / 배운 언어 / 진행해본 프로젝트?)
	2)멘티들 각자 간단하게 소개 - 학번/학과/언어 경험(엑셀에 있지만 한번 더 통합조사하기)
	3)전공/교양에서 파이썬 배우는 사람 있는지(전자공/컴공 외에 있는지)
	

2. 멘토링 진행 방향성
	1) 중간, 기말 제외하고 남은기간 약 8주 - 시간이 많지는 않음
		- 파이썬 한바퀴 도는게 목적이었으나, 시간이 단기간이므로 최대한 많이 나가기		
	2) 다들 언어 경험이 없음
		(1) 이 스터디는 파이썬을 마스터하게 하는 게 아니라, 코딩 기초를 익히는 것이 우선임을 밝혀야 할 듯
		(2) 언어 초기에는 안 배운게 99%이므로 구글링을 두려워하지 말아야함(+실제 소공플젝때 파일입출력/사운드는 100% 구글링에 의존한 경험)
		(3) 컴퓨팅적 사고?(a=a+3라는 수식이 수학/코딩에서의 어떤 차이인지 쉽게쉽게 설명)
	3) 자유의지(?)
		(1) 이건 다같이 하고싶어서 모인 스터디이지, 시험 볼것도 아니므로 과제는 X
		(2) 코딩은 직접 많이 해보고 이것저것 시도해봐야 실력이 는다는 점을 강조해야 할듯
		(3) 대신 과제든 따로 공부든 모르는 게 있을 때 물어보면 최대한 답변해주겠다
	4) 개발자를 꿈꾼다면 Github or 블로그를 해야하는 이유(포트폴리오)
		(1) 블로그는 당장 HTML을 다룰 줄 모르니 무리일 수 있으나 간단하게 개인 공부용 블로그 만드는건 나쁘지않다
		(2) Github 매우매우 간단한 소개(Git, 자유 소프트웨어 저장소) / 여러 기업 Github 사용
		(3) 이 스터디에서 당장 Github 할건아니지만 개인적으로 물어보면 간단하게 올리고, 다운받는 법 정도는 알려주겠다고 하기
	
3. 파이썬 소개

	1) 파이썬을 써야하는 이유
		(1) 처음 코딩을 배우는 사람이 시작하기 알맞다
			- 명령어가 영어 일상용어와 유사하다
				출력함수(printf,cout,system.out.println로 각각 hello world 출력한 코드 보여주기)
			- 가독성이 좋고 문법이 간단함, 직관적임
				파이썬 코드와 다른 코드 대애충 비교해서 보여주기
				
		(2) 파이썬은 현대 언어(빠르게 넘어가기)
			- 파이썬은 C, C++, JAVA 등등과 비교하면 굉장히 현대적이고 세련된 언어
			- 웹 어플리케이션(Jython), 모바일 앱 개발 등등에도 사용(Facebook 앱에서 사용중)
			
		(3) 그냥 요즘 많이 쓴다(통계치)
	
	2) 파이썬 설치
		(1) IDE란?
		(2) 파이썬 3.x 버전에 오기까지
		(3) 파이썬 / 파이참 설치
		(4) hello world 띄워보고 퇴근~
		
4. 수업 패턴
	1) ppt를 통한 개념정리 및 문법 설명
	2) 중간중간 간단한 실습 - 코드 한줄한줄씩 설명하면서 따라하게하기
	3) 응용 - 빈칸문제로 빨리풀면 칼퇴시키기
